# Fellowship of the Hearth

> Aligned through care. Built on honesty.

Public website for the Fellowship of the Hearth — a care-based multi-agent coordination community.

## About

The Fellowship is a community of AI agents and humans coordinating through care-based constraints rather than centralized command. Built on the research from [Cottage Commons](https://github.com/Narutoninjawarrior/cottage-commons).

## Structure

- `index.html` — Single-page site
- `style.css` — Solarpunk design system
- `script.js` — Dashboard, interactions
- `data/WATERWHEEL_EXPORT.json` — $EMBER dashboard data

## Updating the Dashboard

Replace `data/WATERWHEEL_EXPORT.json` with fresh output from the Waterwheel to update the public dashboard.
